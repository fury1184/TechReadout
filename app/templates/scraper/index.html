How Spec Lookup Works (v3.5.1)

TechReadout uses a seed-first, on-demand lookup model:

  1. Seed database — a curated library of common hardware ships pre-populated. Most lookups resolve here, instantly and for free.
  2. Scrape.Do (paid) — proxy fallback for items not in the seed DB.
  3. Open WebUI (optional) — a self-hosted LLM guesses specs when Scrape.Do comes up empty. Always routed to Pending Review for a human check — never auto-accepted.
  4. AI Import — manual catch-all when nothing else works.

Specs are saved to the database on first lookup and cached forever. You only fetch each part once.

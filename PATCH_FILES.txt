TechReadOut 3.5.4 patch files
Apply over TechReadOut 3.5.3.

Purpose:
- Add RAM ECC/non-ECC tracking.
- Add RAM module type tracking: UDIMM/RDIMM/LRDIMM/SODIMM/etc.
- Display RAM ECC/module type in summaries and detail rows.
- Include ECC/module type in lookup API, manual add, AI import, export/import, and Open WebUI prompt.
- Keep Step 6 compatibility logic out of this patch.

Important migration for existing installs:
Run this against the MariaDB database after overlaying files:

  migrations/v3.5.4_ram_ecc.sql

Example from the TRO folder if using the default docker-compose service names:

  docker compose exec -T db mariadb -u techreadout -ptechreadout techreadout < migrations/v3.5.4_ram_ecc.sql

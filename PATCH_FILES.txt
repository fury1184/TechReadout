TechReadOut 3.5.6 Version Hotfix
=================================

Apply over the existing TechReadOut 3.5.6 source tree.

Changed file:
- app/version.py

Fix:
- Restores APP_DISPLAY_VERSION expected by app/__init__.py.

Apply:
1. Extract this archive into /opt/stacks/techreadout and allow it to overwrite app/version.py.
2. Rebuild/recreate the application container:

   cd /opt/stacks/techreadout
   docker compose up -d --build --force-recreate

Verification:

   docker compose exec app python -c "from app.version import APP_NAME, APP_VERSION, APP_DISPLAY_VERSION; print(APP_DISPLAY_VERSION)"

Expected output:

   TechReadOut v3.5.6

If the Compose service is not named app, run:

   docker compose config --services

and substitute the correct service name.
